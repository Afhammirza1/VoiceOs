# VoiceOS 🎙️

An AI-powered Android assistant that lets you control your entire phone using voice commands — open apps, type messages, write emails, and more. Your mobile Jarvis!

## 🛠 Tech Stack

- Android (Kotlin or Java)
- Android SpeechRecognizer API
- Accessibility Service API
- NLP (Google or OpenAI APIs)
- Optional: Offline TTS & ASR

## 🌟 Features

- 📱 Launch any installed app by voice
- 📝 Compose messages or notes by speech
- ✉️ Email dictation and sending
- 🔄 Multi-language support
- 🔒 Works offline for key tasks (with embedded model)

## 🚀 Getting Started

- Clone the project in Android Studio.
- Enable Accessibility & Voice Services.
- Run on physical device for full access.
